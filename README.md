## рЯУМ Problem Statement

Banks need to assess the **creditworthiness of loan applicants** before approving loans.
This project builds a **multi-class classification model** that predicts which risk tier an applicant falls into:

| Tier | Meaning              |
|------|----------------------|
| P1   | Lowest risk вЬЕ        |
| P2   | Low-moderate risk    |
| P3   | Moderate-high risk   |
| P4   | Highest risk вЭМ       |

## вЪЩпЄП Feature Engineering

| Technique       | Applied To           | Purpose                                |
|-----------------|----------------------|----------------------------------------|
| Chi-Square      | Categorical features | Test association with target variable  |
| VIF (вЙ§ 6)       | Numerical features   | Remove multicollinear features         |
| ANOVA (p вЙ§ 0.05)| Numerical features   | Keep only statistically significant    |
